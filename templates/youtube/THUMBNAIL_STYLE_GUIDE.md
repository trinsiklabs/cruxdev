# Thumbnail Style Guide

> Visual standards for video thumbnails. Consistency builds brand recognition and improves CTR.

## Thumbnail Specifications

| Attribute | Value |
|---|---|
| Resolution | 1280 x 720 px (16:9 ratio) |
| File size | Under 2 MB |
| Format | JPG or PNG |
| Safe zone | Keep key elements in center 90% (edges crop on some devices) |
| Text readability | Must be legible at 160 x 90 px (mobile search results) |

## Visual Formula

```
[Background color/image] + [Face or key visual] + [2-4 word text overlay]
```

**Text rules:**
- Maximum 4 words on thumbnail
- Each word should be readable at mobile size
- Outline or drop shadow for contrast against any background
- Do NOT repeat the video title verbatim — thumbnail text should complement, not duplicate

| Content Type | Thumbnail Style | Example |
|---|---|---|

| List/tips | [Number prominent + topic icon] | [Large "7" + topic visual] |
| Live/stream | [Live badge + topic] | ["LIVE" banner + topic] |

## Quality Checklist

Before uploading any thumbnail:

- [ ] Legible at 160 x 90 px (shrink to check)
- [ ] Follows brand color palette
- [ ] No more than 4 words of text
- [ ] High contrast (text is readable against background)
- [ ] Face/expression conveys emotion (if using face)
- [ ] Does NOT duplicate the video title text
- [ ] File size under 2 MB
- [ ] 1280 x 720 resolution
