## 3. Keyword Research Process

### 3.1 Sources

| Source | What It Provides |
|---|---|
| [Ahrefs / SEMrush] | Search volume, keyword difficulty, competitor keywords |
| Google Search Console | Actual queries driving impressions and clicks |
| Google Autocomplete | Long-tail keyword ideas |
| People Also Ask | Question-format keywords |
| Competitor analysis | Keywords competitors rank for that we don't |
| Customer interviews | Language customers actually use |
| Support tickets | Problems customers describe in their own words |

### 3.2 Selection Criteria

| Criterion | Weight | Threshold |
|---|---|---|
| Search volume | Medium | [e.g., ≥100 monthly searches for blog, ≥500 for product pages] |
| Keyword difficulty | High | [e.g., KD ≤ 40 for new sites, KD ≤ 60 for established sites] |
| Business relevance | Highest | [Must relate to our product, audience, or industry] |
| Search intent match | Highest | [Must match the content type we can create for it] |
| Conversion potential | High | [Commercial keywords prioritized over pure informational] |

## 4. Cannibalization Prevention

### Rules

1. **One primary keyword per page.** Never target the same primary keyword on two pages.
2. **Related keywords cluster together.** Secondary keywords go on the same page as their primary.
3. **If two pages compete**, merge them or differentiate (one broad, one specific).
