# On-Page SEO Checklist

> Per-page SEO checklist. Copy this checklist for every page and complete before publishing.

## Page: [Page Name / URL]

**Target keyword:** [Primary keyword]
**Secondary keywords:** [List 2-3]
**Keyword map reference:** [Link to KEYWORD_MAP.md entry]

---

### Title Tag
- [ ] Contains primary keyword (naturally, preferably near the beginning)
- [ ] Length: 50-60 characters (fits in SERP without truncation)
- [ ] Includes brand name (at end: "... | Brand Name")
- [ ] Unique across the site (no duplicate title tags)
- [ ] Compelling — would you click this in search results?

**Title tag:** `[____________]` ([XX characters])

### Meta Description
- [ ] Contains primary keyword (naturally)
- [ ] Length: 120-160 characters
- [ ] Includes a call-to-action or value proposition
- [ ] Unique across the site
- [ ] Accurately describes the page content

**Meta description:** `[____________]` ([XXX characters])

### URL
- [ ] Short, descriptive, contains keyword
- [ ] Lowercase, hyphens as separators
- [ ] No parameters, IDs, or unnecessary folders
- [ ] Matches the page's topic (not just /page-1/)

**URL:** `[/____________/]`

### Headings
- [ ] Exactly one H1 per page
- [ ] H1 contains primary keyword (naturally)
- [ ] H2s organize major sections
- [ ] H3s subdivide H2 sections (proper hierarchy — no skipping levels)
- [ ] Headings are descriptive and scannable
- [ ] At least one H2 contains a secondary keyword

### Content
- [ ] Primary keyword appears in first 100 words
- [ ] Secondary keywords used naturally throughout
- [ ] Content satisfies search intent (what would the searcher want to find?)
- [ ] Content is comprehensive for the topic (competitive with top-ranking pages)
- [ ] Minimum word count met: [e.g., 1,000+ for blog posts, 300+ for product pages]
- [ ] No keyword stuffing — reads naturally
- [ ] Uses related terms and synonyms (semantic SEO)
- [ ] Short paragraphs (2-4 sentences max for web readability)
- [ ] Lists, tables, or other scannable formatting where appropriate

### Images
- [ ] All images have descriptive alt text
- [ ] At least one image alt text includes target keyword (if natural)
- [ ] Images are relevant to content (not decorative stock photos)
- [ ] File names are descriptive (deploy-dashboard.webp, not IMG_3847.webp)
- [ ] Images are optimized (WebP, compressed, responsive srcset)
- [ ] Explicit width/height to prevent CLS

### Internal Links
- [ ] Page links to at least 2-3 other relevant pages on the site
- [ ] At least 2-3 other pages link TO this page
- [ ] Anchor text is descriptive (not "click here")
- [ ] Links are contextually relevant (not forced)
- [ ] No orphan pages (every page reachable from at least one other page)

### External Links
- [ ] Links to authoritative sources where referencing data or claims
- [ ] External links open in new tab (`target="_blank" rel="noopener"`)
- [ ] No broken external links
- [ ] Not excessive (don't link out more than necessary)

### Schema / Structured Data
- [ ] Appropriate schema type applied: [Article / Product / FAQ / HowTo / Organization / WebPage / BreadcrumbList]
- [ ] Schema validates in Google's Rich Results Test
- [ ] Schema matches visible page content (no misleading markup)

### Open Graph / Social
- [ ] og:title set (can differ from title tag for social appeal)
- [ ] og:description set
- [ ] og:image set (1200x630px minimum)
- [ ] og:url set (canonical URL)
- [ ] Twitter card meta tags set (twitter:card, twitter:title, twitter:description, twitter:image)

### Technical
- [ ] Page loads in under 2.5s (LCP)
- [ ] Mobile-friendly (passes Google Mobile-Friendly Test)
- [ ] Canonical tag present and correct (self-referencing or pointing to canonical)
- [ ] No noindex tag (unless intentionally excluded from search)
- [ ] Page is in XML sitemap
- [ ] Page is reachable within 3 clicks from homepage

## Quick Reference: Title Tag Formulas

| Formula | Example |
|---|---|
| [Primary Keyword] — [Benefit] \| [Brand] | "Deployment Automation — Ship Code 10x Faster \| ProductName" |
| [How to / What is] [Keyword] [Year] | "How to Set Up CI/CD in 2026" |
| [Number] [Keyword] [Modifier] | "7 Deployment Best Practices for Startups" |
| [Keyword]: [Expanded Description] \| [Brand] | "CI/CD Platform: Automated Testing and One-Click Deploys \| ProductName" |
