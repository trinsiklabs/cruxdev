## Scoring Guide

| Score | Meaning |
|---|---|
| 1.0 | Fully achieved |
| 0.7 | Strong progress, stretch goal partially met |
| 0.5 | Meaningful progress but fell short |
| 0.3 | Some progress |
| 0.0 | No progress |

**Target:** 0.6 - 0.7 average. If you consistently score 1.0, your OKRs are not ambitious enough.

### Common Pitfalls Checklist

- [ ] Key results are outcomes, not tasks (not "Launch feature X" but "X% of users adopt feature X")
- [ ] Each objective has 2-5 key results (not 1, not 10)
- [ ] Key results are measurable with a number
- [ ] At least one OKR per objective is a stretch (0.7 would be great)
- [ ] OKRs are not performance review criteria (OKRs encourage ambition; performance reviews should not punish missing stretch goals)
