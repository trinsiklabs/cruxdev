## How We Use These

1. **Hiring:** Every candidate is evaluated against values. Skills can be taught; values alignment cannot.
2. **Performance:** Values are part of performance conversations.
3. **Strategy:** When two strategic options are equal on data, values break the tie.
4. **Conflict resolution:** When people disagree, returning to shared values finds common ground.
5. **Brand:** External communications reflect these values.
