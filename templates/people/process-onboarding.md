### DAY 1

**Morning:**
- [ ] Welcome meeting with manager (30 min)
  - Team overview, role expectations, communication norms
- [ ] IT setup and equipment walkthrough
  - Verify all accounts and access working
  - Security training (passwords, MFA, VPN)
- [ ] I-9 verification completed (Section 2 — within 3 business days)
- [ ] Benefits enrollment review with HR
- [ ] Tour of office / virtual workspace walkthrough

**Afternoon:**
- [ ] Meet onboarding buddy — informal introduction
- [ ] Read: Employee Handbook acknowledgment signed
- [ ] Read: Code of Conduct acknowledgment signed
- [ ] Read: Security policies review
- [ ] Team introduction (meeting or async intros)
- [ ] Set up communication tools and test

**End of Day:**
- [ ] Quick check-in with manager (15 min) — how did Day 1 go?
