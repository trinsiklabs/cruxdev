### 6. SECURITY

#### 6.1 Device Security
- Use Company-provided or approved devices only
- Enable full-disk encryption
- Enable screen lock (max {{SCREEN_LOCK_TIMEOUT}} minute timeout)
- Keep OS and software updated
- Enable MFA on all accounts

#### 6.2 Network Security
- Use Company VPN when accessing internal resources
- Do not use public Wi-Fi for sensitive work without VPN
- Secure home Wi-Fi with WPA3/WPA2 and a strong password

#### 6.3 Physical Security
- Lock devices when unattended
- Do not leave Company materials visible in public spaces
- Use privacy screens in public or shared spaces
- Shred sensitive printed documents

#### 6.4 Incident Reporting
Report any security incident, lost/stolen device, or suspected breach immediately to {{SECURITY_CONTACT}}.
