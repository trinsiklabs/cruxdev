## 4. On-Page SEO Checklist

For each content piece:

- [ ] Title tag includes primary keyword (<60 characters)
- [ ] Meta description includes keyword and CTA (<155 characters)
- [ ] URL slug is clean and includes keyword
- [ ] H1 includes primary keyword
- [ ] H2s include secondary keywords naturally
- [ ] Primary keyword in first 100 words
- [ ] Images have descriptive alt text with keywords
- [ ] Internal links to 3-5 related pages
- [ ] External links to 2-3 authoritative sources
- [ ] Schema markup added (FAQ, HowTo, Article as appropriate)
- [ ] Open Graph tags set for social sharing
- [ ] Page loads <3 seconds
- [ ] Mobile-friendly layout
