## Pre-Launch Verification

Every item must be checked before the campaign goes live. Items marked (BLOCKER) will prevent launch if incomplete.

---

### 1. Strategy & Approvals

- [ ] **(BLOCKER)** Campaign brief approved and signed off
- [ ] **(BLOCKER)** Budget approved by finance
- [ ] **(BLOCKER)** Creative approved by brand/marketing lead
- [ ] **(BLOCKER)** Legal/compliance review complete
- [ ] Strategy document finalized
- [ ] Timeline shared with all stakeholders
- [ ] Team RACI confirmed — everyone knows their role
- [ ] Kill criteria defined and agreed upon

### 2. Audience & Targeting

- [ ] Target audience segments defined and documented
- [ ] Email lists segmented and cleaned (bounces removed)
- [ ] Ad platform audiences configured (custom + lookalike)
- [ ] Exclusion lists applied (existing customers, competitors, irrelevant geos)
- [ ] Retargeting audiences created and pixels verified

### 3. Landing Pages & Website

- [ ] **(BLOCKER)** Landing page(s) live and accessible
- [ ] **(BLOCKER)** All links work — no broken links or 404s
- [ ] **(BLOCKER)** Forms submit correctly — test submission received
- [ ] **(BLOCKER)** Mobile responsive — tested on iOS and Android
- [ ] Page load time <3 seconds (test with PageSpeed Insights)
- [ ] Hero copy, images, and CTA match approved creative
- [ ] Thank-you / confirmation page exists and displays correctly
- [ ] Social sharing meta tags (OG image, title, description) correct
- [ ] Favicon present
- [ ] SSL certificate valid (https)
- [ ] 301 redirects configured for any old URLs
- [ ] A/B test variants configured and splitting correctly

### 4. Tracking & Analytics

- [ ] **(BLOCKER)** UTM parameters applied to all campaign links
- [ ] **(BLOCKER)** Google Analytics tracking verified (check real-time view)
- [ ] **(BLOCKER)** Conversion tracking verified — test conversion fires
- [ ] All ad platform pixels installed and verified
- [ ] Tag Manager container published with campaign tags
- [ ] Dashboard created and showing data
- [ ] Attribution model configured
- [ ] Retargeting pixels firing on landing pages
- [ ] Cross-domain tracking verified (if applicable)
- [ ] Event tracking for CTA clicks, form submissions, video plays

### 5. Email

- [ ] **(BLOCKER)** Email content proofread — no typos or errors
- [ ] **(BLOCKER)** Unsubscribe link works
- [ ] **(BLOCKER)** From address is valid and monitored
- [ ] Email renders correctly in Gmail, Outlook, Apple Mail, Yahoo, mobile
- [ ] Subject lines finalized (A/B variants loaded)
- [ ] Preview text set for each email
- [ ] Personalization tokens work ({{first_name}} not broken)
- [ ] Links in emails point to correct UTM-tagged URLs
- [ ] Email sequences/automation triggers configured correctly
- [ ] Send schedule confirmed (dates, times, timezones)
- [ ] Suppression list applied (unsubscribes, bounces, complaints)
- [ ] SPF/DKIM/DMARC configured for sending domain
- [ ] Test email sent to internal team and verified

### 6. Social Media

- [ ] All social posts drafted and approved
- [ ] Posts scheduled in social management tool
- [ ] Images/videos resized for each platform's specs
- [ ] Hashtags researched and appropriate
- [ ] Links in posts have UTM parameters
- [ ] Community managers briefed on campaign and response guidelines
- [ ] Bio/profile links updated (if relevant)

### 7. Paid Media

- [ ] **(BLOCKER)** Ad accounts have sufficient budget/credit
- [ ] Ad creative uploaded and approved by platform
- [ ] Targeting configured correctly (audiences, geos, placements)
- [ ] Bid strategy set per the paid media plan
- [ ] Daily/lifetime budget caps configured
- [ ] Ad scheduling set (if not running 24/7)
- [ ] Ad copy matches landing page messaging
- [ ] Negative keywords applied (search ads)
- [ ] Platform approval received — no policy rejections
- [ ] Frequency caps set (to avoid ad fatigue)

### 8. Content & SEO

- [ ] Blog posts published and indexed
- [ ] Internal links to campaign landing page placed
- [ ] SEO meta titles and descriptions set on all campaign pages
- [ ] Sitemap updated and submitted (if new pages)
- [ ] Content distribution plan confirmed

### 9. Creative & Brand

- [ ] All assets match brand guidelines (logo, color, fonts)
- [ ] All images are high resolution and properly licensed
- [ ] Alt text provided for all images
- [ ] Video captions/subtitles added
- [ ] Accessibility standards met (contrast ratios, screen reader compatible)
- [ ] No placeholder text remaining ("Lorem ipsum", "[TODO]", etc.)

### 10. Legal & Compliance

- [ ] **(BLOCKER)** Disclaimers/terms included where required
- [ ] **(BLOCKER)** Privacy policy reflects campaign data collection
- [ ] Cookie consent banner functional on campaign pages
- [ ] Contest/promotion rules published (if applicable)
- [ ] Copyright notices present
- [ ] Testimonials have proper attribution and permission
- [ ] Claims are substantiated and documented
- [ ] Ad platform policies reviewed — content compliant
- [ ] CAN-SPAM / GDPR / CCPA requirements met for emails

### 11. Team & Operations

- [ ] Launch day communication plan shared with team
- [ ] Monitoring rotation assigned for first 48 hours
- [ ] Escalation path defined (who to contact if something breaks)
- [ ] Customer support team briefed on campaign offer/messaging
- [ ] Sales team briefed (if B2B with sales handoff)
- [ ] Backup plan documented (what if X fails?)
- [ ] Crisis communication plan reviewed

### 12. Technical Infrastructure

- [ ] Server/hosting can handle expected traffic spike
- [ ] CDN configured for campaign assets
- [ ] Form backend / API endpoints tested under load
- [ ] CRM integration verified — leads flow correctly
- [ ] Payment processing tested (if applicable)
- [ ] Email deliverability warmed up (if new sending domain)
