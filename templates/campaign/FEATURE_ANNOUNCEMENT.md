### Tier 1: Major Feature (Net-new capability, top-of-funnel impact)

- [ ] Blog post (long-form with screenshots/video)
- [ ] Email to full user base
- [ ] Social campaign (multi-post, multi-platform)
- [ ] In-app announcement (banner + modal)
- [ ] Changelog entry
- [ ] Press outreach
- [ ] Video demo / tutorial
- [ ] Documentation update
- [ ] Webinar / live demo
- [ ] Partner notification

### Tier 2: Significant Update (Major improvement for existing users)

- [ ] Blog post (medium-length)
- [ ] Email to relevant segment
- [ ] Social posts (2-3 per platform)
- [ ] In-app announcement (banner)
- [ ] Changelog entry
- [ ] Documentation update
- [ ] Video/GIF demo

### Tier 3: Incremental Improvement (Table stakes, bug fixes, polish)

- [ ] Changelog entry
- [ ] In-app tooltip (if UX changed)
- [ ] Documentation update
- [ ] Social post (1 per platform, optional)
