## 5. Production Checklist

### Tech Setup (Test 48 Hours Before)

- [ ] Webinar platform configured and tested
- [ ] Speakers have tested audio, video, screen share
- [ ] Backup internet connection plan for presenters
- [ ] Slide deck / demo environment ready and tested
- [ ] Recording enabled
- [ ] Chat and Q&A features configured
- [ ] Polls / interactive elements set up
- [ ] Breakout rooms configured (if workshop format)
- [ ] Backup presenter identified if primary can't attend

### Day-Of Checklist

- [ ] Start webinar room 15 minutes early
- [ ] Test audio/video one more time
- [ ] Close unnecessary applications on presenter machines
- [ ] Have Q&A moderator ready
- [ ] Have backup slide deck accessible
- [ ] Water/refreshments for presenters
- [ ] "Do not disturb" mode on all devices

---

## 6. Post-Webinar

### Within 24 Hours

- [ ] Send follow-up email to attendees (recording + resources + CTA)
- [ ] Send follow-up email to no-shows (recording + highlights + CTA)
- [ ] Post recording to on-demand page
- [ ] Share highlights/clips on social media
- [ ] Score and route high-intent attendees to sales/onboarding

### Within 1 Week

- [ ] Publish blog post recap or transcript
- [ ] Create short video clips for social (2-3 best moments)
- [ ] Send nurture email with related resource
- [ ] Analyze attendee data for future targeting
